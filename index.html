Three factor authentication

Backend Development

I developed a Three-Factor Authentication System designed to enhance security for banking applications. Built with Django and Python, this system incorporates three layers of user verification: password authentication, a QR code sent via email for one-time use, and biometric verification. Through careful API integration for OTP generation and biometric validation, this project emphasizes a seamless user experience while maintaining high standards of data security and protection. The solution balances convenience with robust security, making it an effective tool for safeguarding sensitive financial data.
